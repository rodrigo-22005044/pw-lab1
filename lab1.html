Programação Web

Na Unidade Curricular de Programação Web abordaremos os princípios basícos associados ao desnvolvedor para a Web, tanto do lado back-end como front-end.

Aprender-se-ão as seguintes teclonolias 🌐:

  • HTML
  • CSS
  • Python
  • Django
  • JavaScript

De forma a consolidar todos os conhecimentos, será feito um laboratório semanal, de entrega obrigatória, de aplicação de matéria lecionada. Irá ser desenvolvido também um projeto.
